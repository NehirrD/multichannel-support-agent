# 🤖 Çok Kanallı Yapay Zekâ Destek Ajanı

Bu proje, **Yapay Zekâ ve Teknoloji Akademisi Real World Challenge** kapsamında geliştirilmiştir.

Amaç; Slack, Web Formu ve E-posta kanallarında çalışan bir yapay zekâ destek ajanı için **güvenlik, politika yönetimi, insan eskalasyonu ve performans ölçümünü** kapsayan bütüncül bir yönetişim (governance) çerçevesi oluşturmaktır.

---

# 🎯 Projenin Amacı

Gerçek hayatta kullanılan yapay zekâ destek sistemlerinde;

- Hassas verilerin korunması,
- Yanlış veya eksik cevapların azaltılması,
- Riskli durumlarda insan müdahalesinin sağlanması,
- Farklı iletişim kanallarında tutarlı davranış sergilenmesi,
- Performansın ölçülebilir hale getirilmesi

hedeflenmiştir.
